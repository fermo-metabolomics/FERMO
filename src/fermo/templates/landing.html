FERMO: Streamlined Molecular Feature Prioritization

FERMO is an app for the processing, analysis, and prioritization of molecules from complex LC-MS/MS data.

FERMO integrates LC-MS/MS data, quantitative biological data, and group metadata in a combined analysis platform. Samples and molecular features are scored for their putative novelty, chemical diversity, and associatedness to quantitative biological data. This allows for pragmatic prioritization of molecular features and samples for further investigation.

FERMO can be used in different modes, which can be accessed by clicking the buttons on the bottom of the page.
More information on FERMO can be found in the README or in the FERMO Wiki.

Processing mode

Standard peak table processing mode.

This mode requires a molecular features peak table and MS/MS data in the MZmine3 format. Optionally, users can also provide files containing grouping metadata, quantitative biological data, and a spectral library for annotation purposes.

Loading mode

Session file loading mode.

This mode requires a previously created FERMO session file in the json format. This can be either created by running data in the peak table processing (standard) mode, or by obtaining a session file from others.
